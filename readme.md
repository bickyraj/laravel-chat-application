## About Chat Application

## Requirements

- Composer
- Nodejs
- Redis Server

## Installation

- clone the project.
- run the following commands
    - composer install
    - npm install
    - php artisan migrate

## Setup The Application

- start redis server
- run the following commands
    - php artisan serve
    - php artisan queue:work
    - node server.js
